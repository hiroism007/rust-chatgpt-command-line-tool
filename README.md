
# ChatGPT Rust Command Line Interface

## Requirements

- openAI API key
  - <https://platform.openai.com/account/api-keys>

## Setup

```bash
export OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

## Build

```bash
cargo build --release
```

## Run

```bash
./target/release/rust-chatgpt-command-line-tool
```
